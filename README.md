# Cl-nica-MVP
Simulación integral de ciberseguridad en entorno clínico utilizando Docker, Kali Linux y herramientas de análisis. Incluye implementación de defensa en profundidad, pentesting, simulación de ingeniería social, gestión de riesgos y respuesta a incidentes basada en estándares NIST y MITRE ATT&amp;CK.
